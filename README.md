# FIACH Automation Base

Base para automatizar el testeo de la GEM FIACH con revisión humana.

Objetivo:
- correr batería de prompts de prueba;
- detectar fricciones;
- comparar respuesta real contra respuesta esperada;
- identificar sobreintervención;
- proponer cambios mínimos;
- dejar reporte;
- preparar PR o cambios revisables;
- ejecutar revisiones periódicas con Codex.

Regla central:
Codex puede auditar y proponer mejoras, pero no debe modificar instrucciones finales sin aprobación humana.
